## 💻 Sobre o projeto

Sistema que aprensenta informações climáticas utilizando a API do site OpenWeather.


---

## ⚙️ Funcionalidades

- [x] Os usuários podem ter acesso as informações meteorológicas de:
  - [x] Temperatura
  - [x] Velocidade do vento
  - [x] Direção do vento
  - [x] Situação do céu
